Windows下virtualenv的使用

  1. 创建名为venv的虚拟环境

    virtualenv venv

  2. 激活：进入venv/Scripts/ 执行activate （也可以一起执行venv/Scripts/activate）

  3. 退出：deactivate

  4. 安装卸载模块 pip install/uninstall xxx==2.0.5

  5. 批量导出和安装库 pip freeze > requirements.txt / pip install -r requirements.txt
